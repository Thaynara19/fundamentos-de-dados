## 📚 Conceitos Trabalhados

### 🔎 Pensamento Analítico

Aprendi que o pensamento analítico envolve **identificar e definir um problema**, para depois solucioná-lo utilizando dados de forma estruturada e organizada.

Percebi que antes de analisar qualquer informação, o mais importante é entender corretamente qual problema precisa ser resolvido.

---

### 📊 Comunicação de Insights

Entendi que os analistas de dados utilizam diferentes formas de visualização para comunicar resultados, como:

* Gráficos
* Mapas
* Diagramas

Essas visualizações ajudam outras pessoas a compreender rapidamente os insights gerados pelos dados.

---

### 🧩 Pensamento Orientado a Detalhes

O exercício mostrou que planejar ações considerando cada etapa necessária — como no exemplo de uma viagem planejada com paradas estratégicas — representa o pensamento orientado a detalhes.

Esse tipo de pensamento é essencial para transformar planejamento em execução eficiente.

---

### 📈 Análise de Lacunas (Gap Analysis)

Aprendi que a análise de lacunas é utilizada para:

* Avaliar como um processo funciona atualmente
* Identificar o que precisa melhorar
* Definir um estado futuro mais eficiente

Esse conceito é muito utilizado para melhoria contínua dentro das empresas.

---

## 💭 Meu Aprendizado

Cada atividade prática tem ajudado a desenvolver uma visão mais estratégica sobre análise de dados.
